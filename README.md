# BATMAN's AI Personal Assistant

A local, privacy-first AI assistant powered by **Ollama** and **Llama 3.2**, built with Streamlit. This project brings personalized AI interactions with multiple personality personas and Gmail integration.

## 🎯 Features

- **Multiple AI Personalities**: Choose from different personas (Joker, Alfred, Robin, Catwoman) that adapt their responses based on the selected personality
- **Local LLM Processing**: Uses Ollama with Llama 3.2 for fast, private inference—no cloud dependencies
- **Gmail Integration**: Check and summarize your unread emails directly in the chat
- **File Upload**: Upload `.txt` files to provide additional context for conversations
- **Chat History**: Persistent chat history within each session with the ability to clear it anytime
- **Real-time Responses**: Get instant AI responses with visual indicators
